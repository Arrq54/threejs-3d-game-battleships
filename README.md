## Installation

Clone project with git and install modules

```bash
git clone https://github.com/Arrq54/threejs-3d-game-battleships.git
cd threejs-3d-game-battleships
npm i
```

Run the app



```bash
node server.js
```
You may now open your browser and go to localhost:3000.
